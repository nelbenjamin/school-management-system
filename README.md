# EduManage — School Management System

A full-stack, role-based school management platform built for teachers, administrators and principals.

## Features
- 🔐 Role-based access control (Teacher / Admin / Principal) with JWT authentication
- 📊 AI-powered grade analytics and personalised learner feedback
- 🧮 Custom CA & final mark formula calculator
- 📄 Professional Excel, PDF and Word report generation
- 🔔 Real-time notifications via Socket.io
- 💾 Persistent accounts — continue where you left off

## Tech Stack
**Frontend:** React, Vite, Framer Motion, Recharts, Socket.io-client  
**Backend:** Node.js, Express, MySQL, Socket.io, JWT, bcrypt  
**Exports:** xlsx, jsPDF, docx

## Setup
1. Clone the repo
2. Set up backend `.env` with your MySQL credentials
3. Run `npm run dev` in both `/backend` and `/frontend`
